# Debunk

## なにこれ
Fiddler ライクな（感じを目指している） HTTP Debugging Proxy です。  
`node-webkit` を用い、クロスプラットフォームに対応します。  

## 進捗
現在、リクエストを止めて書き換える、レスポンスを止めて書き換えることが出来ます。  
デザインはまだまだです。  
デフォルトの設定では、8888番で Listen しています。

## 実装する機能
- （JavaScript による）カスタムルール
- HAR でのインポート・エクスポート
- 表示のカスタマイズ
